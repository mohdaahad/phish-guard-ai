# Quick Start Guide

## Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Overview

This Next.js website replicates the SOL9X landing page from your Figma design with:

- ✅ Responsive header with mobile navigation
- ✅ Hero section with main heading
- ✅ Industry expertise section
- ✅ Work showcase section
- ✅ Power engineering section
- ✅ Why Choose Us section
- ✅ Insights section
- ✅ Footer with SOL9X branding

All components are built with:
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Fully responsive design

## Customization

To match the exact Figma design:
1. Replace placeholder icons with actual images from Figma
2. Adjust colors in `tailwind.config.ts` to match Figma color palette
3. Update fonts in `globals.css` if different from default
4. Add actual images to `/public` folder and reference them in components
